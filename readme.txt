== Description ==

*Extends the default Pages widget with more options:*

* Display content
* Display thumbnail
* Configure thumbnail size
* Choose the Post Type to display

[Demo](http://www.osi.lv/)

== Installation ==

1. Upload `pages-widget` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
